# Spam Detection ML

## Description
Détection de messages spam avec Machine Learning (Naive Bayes, Logistic Regression, Linear SVM) et interface Streamlit.

## Structure du projet
- `data/` : dataset prétraité
- `models/` : modèles et vectorizer sauvegardés
- `src/` : scripts Python (train.py et app.py)
- `notebooks/` : exploration, prétraitement et modélisation
- `requirements.txt` : dépendances Python
- `.gitignore` : fichiers à ignorer
